🔥 Firebase конфігурація

1. Зайдіть на console.firebase.google.com

2. Створіть проєкт → Додайте веб-додаток

3. Build → Realtime Database → Create database (регіон: europe-west1)

4. Rules: { "rules": { ".read": true, ".write": true } }

5. Відкрийте файл firebase-config.js та вставте конфіг зі свого проєкту
